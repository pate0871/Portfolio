COVID Screening & Tracking App

This was a group project completed during 3rd semester for Object Oriented Programming with Design Patterns course. We were required to develop a web application using software development life cycle and using OOP design patterns. Event organizers or office managers could use this application to screen attendee beforehand. We programmed in roles, and every role had difference access levels. Admin or team owner had access to all the attendee's results for their team and attendees could only view their results.
